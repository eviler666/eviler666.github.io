博客搭建教程

发表于2021-12-12|更新于2021-12-17|技术
|阅读量:

最近搭了一个博客，使用github+hexo的框架，以下是极其精简的步骤，嘿嘿

git

github账号

  • 注册自己的github账号
  • 注册一个名为“你的用户名.github.io”的仓库，例：你的github用户名是a123，仓库名就为：“a123.github.com”

git软件

  • 下载并安装git软件

ssh配置

  • 打开Git bash终端，
  • 输入以下命令，连续3次回车，在用户目录下，会找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容
    ssh-keygen -t rsa -C "邮件地址"
  • 回到你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key，其中标题随便填，ssh key粘贴入刚复制的内容，保存即可
  • 检测是否成功，输入以下命令（不用更改邮箱地址）
    ssh -T git@github.com
  • 如果提示Are you sure you want to continue connecting (yes/no)?，输入yes，
  • 然后会看到：Hi liuxianan! You’ve successfully authenticated, but GitHub does not provide shell access.
  • 这代表已经成功！
  • 配置用户名和邮箱
    $ git config --global user.name "liuxianan"// 你的github用户名，非昵称
$ git config --global user.email  "xxx@qq.com"// 填写你的github注册邮箱

以下的代码均建议在git bash终端内执行

Node.js

  • 下载并安装node.js
    node -v	#查看node版本
npm -v	#查看npm版本
  • 安装淘宝的cnpm 管理器
    npm install -g cnpm --registry=http://registry.npm.taobao.org	
cnpm -v	#查看cnpm版本

Hexo

  • 安装并初始化hexo
    cnpm install -g hexo-cli    #安装hexo框架
hexo -v   #查看hexo版本

mkdir blog    #创建blog目录
cd blog   #进入blog目录

hexo init   #生成博客 初始化博客
hexo s    #启动本地博客服务

http://localhost:4000/	#本地访问地址

hexo clean    #清理
hexo g    #生成
hexo s    #启动本地服务
  • 安装hexo-deployer-git插件
    cnpm install --save hexo-deployer-git   #在blog目录下安装git部署插件
  • 配置博客根目录下的_config.yml 
    # Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repo: https://github.com/a123/a123.github.io.git    #示例用户名：a123
  branch: main
  • 部署到github仓库
    hexo clean    #清理一下
hexo g    #生成
hexo d    #部署到Github仓库

https://a123.github.io/   #访问这个地址可以查看博客
  • 克隆主题
    git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly    #下载butterfly主题到本地
  • 修改hexo根目录下的 _config.yml 文件 ： theme: butterfly
    hexo clean    #清理一下
hexo g    #生成
hexo d    #部署到Github仓库

https://a123.github.io/   #访问这个地址可以查看更改主题后的博客

Butterfly

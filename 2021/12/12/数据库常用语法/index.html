数据库常用语法

发表于2021-12-12|更新于2021-12-17|技术
|阅读量:

记录一下常用的数据库语法

oracle

  • 判断后边是否在前边里面，是为1，否为0

    select INSTR(';'||'35;52;117'||';',';'||35||';')>0 from dual

  • 多行转一行

    select listagg(b.note,',') within group(ORDER BY IBM) from lchtrsgsqbd a,txtdm b where b.fldm = 'GRSRLY' and instr(';'||a.xtccly||';',';'||b.ibm||';')>0 and a.id = 100376

  • 一行转多行

    -- 情况一
SELECT distinct REGEXP_SUBSTR(t.B, '[^,]+', 1, level) fkId, t.* 
from QQQ2 t 
connect by level <= regexp_count(t.B, ',') + 1
-- 情况二
SELECT distinct REGEXP_SUBSTR(t.B, '[^,]+', 1, level) fkId, t.* 
from QQQ2 t 
connect by level <= regexp_count(t.B, ',') + 1

  • 字符串函数

    select substr(13703959212,1,3)||'00000'||substr(13703959212,-3,3) from dual
select replace(13703959212,substr(13703959212,4,5),'00000') from dual

  • 日期函数

    to_number(to_char(to_date(b.jzrq,'yyyymmdd')+31,'yyyymmdd'))	--数字型日期转换

  • oracle 归档模式空间不足

    删日志

    先切到oracle用户下

    然后rman target /进入rman

    最后删除命令：delete archivelog until time ‘sysdate-7’;

    如果是ADG部署，使用这条命令删除：DELETE noprompt force ARCHIVELOG ALL COMPLETED BEFORE ‘SYSDATE-7’;

    查看路径

    su - oracle

    sqlplus / as sysdba

    archive log list

Hana

  • 开窗

    row_number() over(partition by recordID ORDER BY case when k.empno is null then 'G' else grade end) num		-- 相等的值不进行区分
rank() over()		-- 相等的值排名相同，序号从1到n不连续
dense_rank() over()		-- 相等的值排名相同，序号从1到n连续
